Docker Compose简介

Docker Compose是一个用于定义和运行多容器Docker应用程序的工具。通过Compose，您可以使用YAML文件来配置应用程序的服务，然后使用一个命令创建并启动所有服务。它极大地简化了容器编排工作，特别适合开发、测试和CI工作流程。
